# 💰 Expense Tracker

A responsive Expense Tracker web application built using **HTML, CSS, and JavaScript** that helps users track their income and expenses, calculate their current balance, and maintain a transaction history.

## 🌐 Live Demo

🔗 https://expense-tracker-nirisha.netlify.app

## 📂 GitHub Repository

🔗 https://github.com/Nirisha22/Expense-Tracker

---

## 📌 Features

- 💵 Add income transactions
- 💸 Add expense transactions
- 📊 Automatically calculates total balance
- 📈 Displays total income and total expenses
- 📝 Maintains transaction history
- ❌ Delete transactions
- 📱 Responsive design for desktop and mobile
- ⚡ Instant balance updates without refreshing the page

## 🚀 How to Run Locally

1. Clone the repository

```bash
git clone https://github.com/Nirisha22/Expense-Tracker.git
```

2. Open the project folder

```bash
cd Expense-Tracker
```

3. Open `index.html` in your browser

OR

Run using VS Code Live Server.
